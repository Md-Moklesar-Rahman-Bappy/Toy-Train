# Toy Train 🚂

This project demonstrates smooth animations, responsive design, and easy deployment with GitHub Pages.

## 📦 Installation & Setup
To run this project locally:

1. **Clone the repository:**
```bash
   git clone https://github.com/Md-Moklesar-Rahman-Bappy/Toy-Train.git
```
2. **Navigate into the project folder:**
```bash
   cd Toy-Train
```
3. **Open index.html in your browser.**

---

## 🌍 Deployment
This project is deployed using GitHub Pages.
To deploy your own version:- 
1. Push your code to a GitHub repository.
2. Go to Settings > Pages.
3. Select the branch (main or master) and root folder.
4. Save, and your site will be live at:
https://<username>.github.io/<repository>/
